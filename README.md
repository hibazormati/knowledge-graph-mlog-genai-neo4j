# 🕸️ Knowledge Graph for Graph Machine Learning in the Era of Generative AI

## Overview

This project presents a **Knowledge Graph** built in **Neo4j** to model the ecosystem of **Graph Machine Learning (GML)** in the era of **Generative AI**.

The graph was designed from the content of the **MLoG-GenAI Workshop (ACM SIGKDD)** and represents the relationships between graph learning techniques, application domains, technical challenges, and emerging opportunities introduced by foundation models.

Rather than implementing Graph Neural Networks, this project focuses on **knowledge representation**, **graph modeling**, and **graph database technologies**.

---

## Objectives

- Model Graph Machine Learning concepts as a Knowledge Graph
- Represent relationships between AI technologies
- Build the graph using Neo4j
- Query the graph using Cypher
- Produce interactive graph visualizations
- Provide an educational resource for exploring Graph AI

---
